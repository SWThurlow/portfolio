About this project

Where I started

This project started off as a simple game using prompts and displaying outcomes in the console. While this allowed the making of the game to be not too difficult to start with having prompts pop up and having to view outcomes in the console is far from a user friendly game to play.

How it changed

While I had made a working game it's impractical display led to my choosing to make a basic interface to play the game through. Which if I start exploring making digital art work I may make something so that there can be images for each of the actions to make the interface slightly less spartan.

The javascript interaction with the interface lead to some more challenging problems with the code such as the game log being able to reach slightly outrageous lengths due to my not incorporating a way to reset the game at first.

How I improved

About this project

Where I started

This is another project from the Odin projects syllabus. That meant that the main goal with this project was to help develop more of an understanding of object constructors and classes when the code was later refactored.

How it changed

The main way this project changed was the moving to the use of classes. However I did also spend some time playing around with various layout and colour schemes, something I should have possible have done after more of a break given how much classes had me scratching my head.

How I improved

That I am not a massive fan of classes and much prefer factory function would be the first important thing this project taught me. The second one being that I need to build many more things and try to focus on improving my skills with designing layouts and picking colours before I get good at making things that look good without copying design ideas from other people.
